CUBE POTATO BY SAMSEEN

This resource pack changes the default flat potato item model to a 3D cube-shaped potato with rounded corners, inspired by the potato prop shown in the Minecraft movie, as featured in Mumbo Jumbo's behind the scenes tour YouTube short.

Created by: SamSeen
Compatible with: Minecraft 1.21.1 and newer

Installation:
1. Place this folder in your Minecraft "resourcepacks" folder
   - Windows: %appdata%\.minecraft\resourcepacks
   - macOS: ~/Library/Application Support/minecraft/resourcepacks
   - Linux: ~/.minecraft/resourcepacks
2. Launch Minecraft
3. Go to Options > Resource Packs
4. Move the "Cube Potato by SamSeen" to the right side (Selected Resource Packs)
5. Click "Done"

Features:
- 3D cube-shaped potato model with rounded corners
- Custom potato texture
- Rotated at a 75-degree angle for better visibility
- Authentic movie-inspired design

Enjoy your new cube potatoes!
